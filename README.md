# LearnSnap - تطبيق توليد الاختبارات بالذكاء الاصطناعي

> حوّل صور صفحات الكتب المدرسية إلى اختبارات تفاعلية فوراً

---

## نظرة عامة

LearnSnap هو تطبيق تعليمي عربي موجه للهواتف المحمولة يحول صور صفحات الكتب المدرسية إلى 20 سؤال اختبار متنوع باستخدام الذكاء الاصطناعي المتقدم مع نظام تحقق متعدد المراحل.

---

## المميزات الرئيسية

### توليد الاختبارات
- **رفع حتى 20 صفحة** في الاختبار الواحد
- **20 سؤال**: 17 اختيار متعدد + 3 صح/خطأ
- **6 مراحل تحقق** لضمان جودة الأسئلة
- **3 نماذج ذكاء اصطناعي** للتحقق من الإجابات

### المصادقة
- **تسجيل بالبريد الإلكتروني** مع التحقق
- **Google OAuth 2.0** (اختياري)
- **جلسات آمنة** لمدة 30 يوم

### الدفع
- **نظام الدفع لكل صفحة** عبر Paylink
- **دعم mada, Visa, Mastercard, Apple Pay, STC Pay**
- **مكافأة المستخدمين الأوائل**: أول 30 مستخدم يحصلون على 50 صفحة مجاناً

## نقاط API الرئيسية

| Method | Endpoint | الوصف |
|--------|----------|-------|
| POST | `/api/quiz/create` | إنشاء اختبار من الصور |
| GET | `/api/quiz/:id` | جلب اختبار بالمعرف |
| POST | `/api/auth/register` | تسجيل مستخدم جديد |
| POST | `/api/auth/login` | تسجيل الدخول |
| GET | `/api/auth/google` | Google OAuth |
| POST | `/api/credits/check` | التحقق من الرصيد |
| GET | `/api/health` | فحص صحة الخادم |
